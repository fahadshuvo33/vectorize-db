# Supabase SQL Migrations

Write your SQL migrations here and run them manually in Supabase SQL Editor.

## How to Use

1. Create SQL file: `supabase/migrations/002_your_migration.sql`
2. Write your SQL
3. Copy and paste into Supabase Dashboard > SQL Editor
4. Run it

## Initial Schema

See `001_initial_schema.sql` for the base tables (profiles, social_logins) and RLS policies.
